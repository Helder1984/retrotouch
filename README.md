# RetroTouch

Adaptive mobile controls for retro handheld RPGs.

## Vision

RetroTouch aims to transform classic handheld RPGs into native-feeling mobile experiences through adaptive touch controls and contextual UI systems.

## Initial Goals

- Integrate with mGBA
- Create touch-based virtual controls
- Implement contextual RPG interfaces
- Support adaptive overlays and gestures
